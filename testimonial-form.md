# KiddoKlub — Testimonial Capture System
> Send after every rental. Collect real reviews. Update the website with genuine social proof.

---

## Google Form Structure

Create a Google Form with these fields:

1. **Your Name** (Short answer, required)
2. **Your Child's First Name** (Short answer, optional)
3. **Which area of Doha was your event?** (Short answer — e.g., "West Bay", "The Pearl")
4. **What was the occasion?** (Dropdown: Birthday party / Playdate / School event / Other)
5. **Which theme did you choose?** (Dropdown: Neutral Nest / White Wonderland / Color Pop)
6. **How would you rate your experience?** (Linear scale: 1-5 stars)
7. **What did you love most about KiddoKlub?** (Paragraph, required — this is the testimonial)
8. **Would you recommend KiddoKlub to other parents?** (Multiple choice: Yes / Maybe / No)
9. **Can we use your review on our website and Instagram?** (Multiple choice: Yes, with my first name / Yes, anonymously / No)
10. **Upload a photo from the event** (File upload, optional)

**Form settings:**
- Turn on email notifications so Ivan gets alerted immediately
- Set confirmation message: "Thank you! Your feedback means the world to us. Don't forget — refer a friend and you both get QAR 200 off! 🎪"

## How to Use Collected Testimonials

Once you have 3+ real testimonials with permission:

1. **Update index.html** — Replace the 3 current (fabricated) testimonials with real ones
2. **Instagram stories** — Screenshot the review, add to "Reviews" highlight
3. **Feed posts** — Create a quote graphic with the testimonial text
4. **WhatsApp Status** — Share as social proof

### Format for Website

```html
<div class="testimonial-card">
  <div class="stars">★★★★★</div>
  <p>"[THEIR EXACT WORDS]"</p>
  <div class="testimonial-author">— [First Name] [Last Initial].</div>
  <div class="testimonial-location">[Occasion], [Area]</div>
</div>
```

## Testimonial Collection Checklist

After EVERY rental:
- [ ] Send thank-you + form link (2-4 hours after collection)
- [ ] Follow up if no response (3 days later)
- [ ] If they reply with a testimonial in chat (not the form), ask: "Can we use this on our website?"
- [ ] Save all testimonials in a running list
- [ ] Once you have 3+: update the website
- [ ] Create Instagram quote graphic from best ones

## Bonus: Quick Review Request (For Chat-Based Reviews)

If someone messages you saying they loved it but won't fill out a form:

```
That's so kind! Would you mind if I used that as a review on our website?
I'd just say: "[their words]" — [First Name], [Area]

Totally fine if not!
```

Most people say yes when you make it this easy.
