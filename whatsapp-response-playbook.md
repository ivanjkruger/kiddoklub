### Labels (tag every conversation)
- 🟢 **New Lead** — just inquired
- 🟡 **Quoted** — sent pricing, waiting for reply
- 🔴 **Hot** — ready to book, needs date/payment
- 🔵 **Booked** — confirmed + paid
- ⚫ **Done** — event completed
- ⬜ **Follow Up** — went quiet, needs nudge

## Response Decision Tree

```
Customer messages →
│
├─ "How much?" / Pricing question
│   → Send: PRICING REPLY
│
├─ "What do you have?" / General inquiry
│   → Send: OVERVIEW REPLY
│
├─ "Are you available on [date]?"
│   → Check calendar → Send: AVAILABILITY REPLY
│
├─ "I want to book" / Ready to book
│   → Send: BOOKING CONFIRMATION
│
├─ Quote builder message (auto-generated from website)
│   → Send: QUOTE FOLLOW-UP
│
├─ "Do you deliver to [area]?"
│   → Send: DELIVERY REPLY
│
├─ "Is it safe for toddlers?"
│   → Send: SAFETY REPLY
│
├─ Wants to cancel / reschedule
│   → Send: CANCELLATION REPLY
│
├─ Referral inquiry
│   → Send: REFERRAL REPLY
│
├─ No response after 24 hours
│   → Send: FOLLOW-UP #1
│
├─ No response after 3 days
│   → Send: FOLLOW-UP #2
│
└─ Post-event
    → Send: THANK YOU + REFERRAL ASK
```

## Response Time Rules

| Time of Day | Target Response Time |
|-------------|---------------------|
| 8 AM - 10 PM | Under 5 minutes |
| 10 PM - 8 AM | Away message auto-replies → respond first thing in AM |
| During event setup/collection | Team member responds OR away message |

**Golden rule**: The faster you reply, the more you book. Every minute of delay = higher chance they message a competitor.
